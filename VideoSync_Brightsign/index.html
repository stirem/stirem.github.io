- Handbrake H.265 3840x2160 29.97fps HEVC works.
- Important that the resolution settings on file is the same as the screen.

2. Format SD-card using «SD-formatter»
3. Copy "autorun.brs" from MASTER folder and videofile to to master SD-card. Copy "autorun.brs" from CLIENT1 and videofile to client1 sd-card
